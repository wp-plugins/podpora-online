== Description ==

Online podpora pro českou komunitu. Naleznete zde překlady pluginů, šablon a mnoho dalšího.

Naleznete zde již přeložené:

* Čeština buddypress
* Shortcodes Ultimate překlad
* Subscribe2 čeština
* WooCommerce čeština
* Connections čeština
* Podpora online
* Paid Memberships Pro čeština
* CubePoints čeština  ( nově )
